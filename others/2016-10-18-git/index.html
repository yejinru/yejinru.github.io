Git

Posted on 2016-10-05 | In others |

本文记录了git和svn的常用命令。

1. git

  1. 创建项目：

  1. 下载代码：

    1. 下载代码：git clone ${xxx.git}

    2. 下载指定分支的代码：git clone -b ${branch_name} ${xxx.git}

    3. 根据commit_id下载文件：git checkout -b ${branch_name} ${commit_id}

  2. 分支管理：

    1. 查看分支：git branch

    2. 创建分支：git branch

    3. 切换分支：git checkout ${branch_name}

    4. 创建+切换分支：git checkout -b ${branch_name}

    5. 删除本地分支：git branch -D ${branch_name}

    6. 删除远程分支：git push origin :${branch_name}

    7. 合并某分支到当前分支：git merge ${branch_name}

    8. 根据commit_id进行merge：git merge ${commit_id}

  3. 开发

    1. 查看当前文件和文件夹状态：git status

    2. 查看diff：git diff

    3. 拉取代码：git pull origin ${banch_name}

  4. 上传代码：

    1. 添加文件：git add ${file_name}

    2. 添加文件夹：git add ${folder_name}

    3. 提交：git commit -m “${your_commit_note}”

    4. 推到远程目录：git push origin ${branch_name}

  5. tag：

    1. 打tag：git tag -a ${tag_name} -m “${tag_message}”

    2. 推tag到远程仓库：git push origin ${tag_name}

    3. 删除tag：git push origin –delete tag ${tag_name}

  6. 下载相关依赖的子模块：

    git submodule init

    git submodule update

  7. 其它：

    1. 查看当前分支最近的提交记录：git log -2

    2. 查看最近一次的提交信息：git log -n 1 -p

    3. 查看指定的commit的内容：git show commit_id

    4. 添加全局密码：

      eval $(ssh-agent)

      ssh-add

2. svn

  1. svn co ${src_url}

  2. svn add ${file_to_commit}

  3. 创建分支：svn copy ${src_url} ${dst_url} -m “your_commit_note”

  4. 合并分支：svn merge ${branch_url}

  5. 查看分支信息：svn log -v --stop-on-copy

  6. 切换分支：svn switch ${branch_url}
