Mac使用技巧

发表于 2016-09-24 | 分类于 mac

I worte this article for my gf :)

1. 触摸板

  1. 双手指上下滑动：上下拉动页面

  2. 双手指左右滑动：前进后退（浏览器上）

  3. 三手指上下滑动：显示所有打开的程序和界面

  4. 三手指左右滑动：切换到另外一个最大化的界面

  5. control + 点击：相当于windows右键

2. 快捷键

2.1. 浏览器(chrome)

  1. 刷新：command + r

  2. 查找：command + f

  3. 恢复上一次关闭的tab：command + shift + t

2.2. 通用（部分源于个人设置，建议看看【系统偏好设置-键盘-快捷键】

  1. 关闭当前窗口：command + w

  2. 关闭整个程序：command + q

  3. 程序之间切换：command + tab

  4. 保存：command + s

  5. 撤销：command + z

  6. 拷贝：command + c

  7. 剪切：command + x

  8. 全选：command + a

  9. 删除文件或文件夹：command + delete

  10. 屏幕截屏（个人设置）：command + shift + a

  11. 屏幕截屏并保存到文件（个人设置）：command + shift + s

  12. 切换输入法（个人设置）：command + p

  13. delete：

    • delete：向前删除

    • fn + delete：先后删除

    • …

3. 命令行操作

  1. ls: 列出当前目录的文件及文件夹

    • ls：列出当前目录的文件及文件夹

    • ll：列出当前目录的文件和文件夹的详细信息

    • ls folder：列出folder的信息

    • ls file：列出file的信息

    • …

  2. pwd：打印当前所在目录

  3. cd：进入文件夹

    • cd：进入到home目录

    • cd ~：进入到home目录

    • cd -：进入到上一次所在的目录

    • cd folder：进入到folder目录

    • cd folder/subfolder：进入到folder目录下的subfolder子目录

    • …

  4. history：列出所有历史操作命令

  5. control + r: 查看历史操作命令

  6. tab：可列出上一次相似命令

  7. cat file：把file的信息在屏幕中打印出来

  8. grep：根据过滤条件从文本信息中筛选出文本信息

    • grep “condition” file：从file中查找带有condition字符串的行

    • grep -r “condition” folder：从folder文件夹下的所有文件查找带有condition字符串的行，condition支持正则表达式

    • grep -E “condition1|condition2” file：从file中查找符合condition1或condition2的行，condition1和condition2支持正则表达式

    • …

  9. touch file：创建空文件file

  10. mkdir：

    • mkdir folder：创建文件夹folder

    • mkdir folder/subfolder -p：当folder不存在时，支持递归创建folder和folder的子文件夹subfolder；当folder存在时，创建subfolder

    • …

  11. rm（慎用，一旦删除，文件或文件夹不可恢复）：

    • rm file：删除文件file

    • rm file*：删除所有前缀为file的文件

    • rm folder -rf：递归删除文件夹folder及文件夹folder下所有子文件夹和所有子文件

    • …

  12. clear：清空屏幕

  13. control + a：在命令行模式下，返回到命令行首

  14. control + e：在命令行模式下，返回到命令行末

  15. control + c：停止当前操作

  16. vim：自行上网学习

  17. tree：

    • tree -l folder：列出folder下所有目录、子目录、文件

    • tree -L 1 folder：只列出folder目录下第1层子目录和文件

    • …

  18. kill：

    • kill -9 MATLAB：强制杀掉含有MATLAB的进程

    • …

  19. etc…
